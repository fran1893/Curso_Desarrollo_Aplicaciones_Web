PARTE 1 (5 puntos): subir una página web a un servidor

Nota : si por problemas técnicos de la conexión no pudiera realizarse esta prueba habrá que hacer necesariamente la OPCIÓN 2

Sube una página web en formato HTML a uno de los dos servidores en los que hemos creado cuenta (CDMON o InfintyFree).

En la página debe constar solo como título h1 tu nombre de esta forma:

TRES primeras letras de tu primer nombre + TRES primeras letras de tu primer apellido.

Ejemplo 1: alumno JUAn Ramón JÍMenez -> JUA+JIM

Nota: por motivos de privacidad no pondremos el nombre completo real

La página se llamará alumno+el identificador anterior. Ejemplo:alumno-juajim.html

Hay que hacer las siguientes capturas de pantalla (con la herramienta recortes de Windows, por ejemplo):

1. Cuando accedes al servidor

2. Antes de entrar al panel de control

3. En el panel de control

3. Cuando hayas subido el fichero html

4. La visualización de la página desde tu sitio web
