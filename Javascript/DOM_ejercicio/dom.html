Document Object Model

1. Definiciones

El DOM es...

la estructura del documento HTML, desarrollada mediante un API para interactuar con los nodos de una página web.

Aunque el desarrollador la codifica con etiquetas de apertura y cierre que se pueden anidar (como cajas que contienen otras cajas), para el navegador se trata de una estuctura jerárquica, similar a un árbol genealógico.

El objeto window...

representa la ventana o pestaña actual.

El objeto document...

es donde se carga la página web. Está contenido en window.

Nodes son...

todos los objetos que contiene el documento: etiquetas(tags), atributos y texto.

Elements son...

los diferentes tipos de nodo: HTMLElement, HTMLTableElement, etc.

Un nodeList es...

un pseudo-array de elements, es decir, se trata de una colección de ítems que a pesar de no ser exactamente un array, se les pueden aplicar algunos de los métodos propios de los arrays, como los bucles.

2. Selectores

document.getElementsByTagName('nombre_de_la_etiqueta')

Retorna los nodos segun su etiqueta (h1, p, etc.).

Se aconseja utilizar querySelectorAll() en su lugar.

document.getElementsByClassName('nombre_de_la_classe')

Retorna los nodos segun el nombre de su clase.

Se aconseja utilizar querySelectorAll() en su lugar.

document.getElementById('nombre_del_id')

Retorna el nodo segun el nombre de su id.

Para seleccionar ids es la forma recomendada, antes que querySelector('#nombre_de_id') ya que ofrece mejor rendimiento.

document.querySelector('nombre_del_elemento')

Retorna el primer nodo segun el nombre de indicado entre comillas. Si no existe devuelve null.

Cuando se trate de un id o una clase hay que incluir al principio los caracteres # (almohadilla) o . (punto), respectivamente.

document.querySelectorAll('nombre_de_los_elementos')

Retorna los nodos según el criterio especificado: etiquetas, clases o combinaciones entre ellas.

Nota: Cuando se trate de un id o una clase hay que incluir al principio los caracteres # (almohadilla) o . (punto), respectivamente.

3. Ejemplos de selecciones

Seleccionar todos los títulos h2

let ejemplo = document.querySelectorAll('h2');

Seleccionar todos los títulos h2 y h3

let ejemplo = document.querySelectorAll('h2, h3');

Seleccionar todos los títulos h2 y h3 que tienen clase 'ejemplos'

let ejemplo = document.querySelectorAll('h2.ejemplos, h3.ejemplos');

Seleccionar el primer div contenido en la section con clase 'teoria' y poner el color de su texto a rojo.

let ejemplo = document.querySelector('section.teoria div');

ejemplo.style.color = 'red';

Seleccionar todos los div contenidos en la section con clase 'teoria'

let ejemplo = document.querySelectorAll('section.teoria div');

Seleccionar todos los div cuyo padre es la section con clase 'teoria' y poner su color de fondo en amarillo.

let ejemplo = document.querySelectorAll('section.teoria >div');

ejemplo.forEach(item => { item.style.backgroundColor = 'yellow'; })

4. DOM traversing

Dom traversing se refiere a recorrer el DOM de forma contextual, a partir de una selección previa, mediante las relaciones de parentesco entre los elementos: padres, hijos, hermanos, pero también es posible, abuelos, nietos...

parentNode

Devuelve el nodo padre. Ejemplo:

const nodo = document.querySelector('.opcionesMenu');

const nodoPadre = nodo.parentNode;

console.log('nodo padre: ', nodoPadre);

Devuelve:

<section id="teoria"></section>

parentElement

Devuelve el elemento padre. Ejemplo:

const nodo = document.querySelector('.opcionesMenu');

const elementoPadre = nodo.parentElement;

console.log('elemento padre: ', elementoPadre);

Devuelve:

<section id="teoria"></section>

En esto caso el resultado es el mismo que con parentNode.

childNodes

Devuelve los nodos hijos si existen. Ejemplo:

var nodeList = elementNodeReference.childNodes;

Al primer nodo se le asigna el índice 0.

Como se trata de nodos devuelve etiquetas y también textos si hay.

firstChild

Devuelve el primer nodo hijo. Ejemplo:

En HTML:

<ul>

<li>Soy el primer hijo<li>

<li>Soy el último hijo<li>

</ul>

En javascript:

const ul = document.querySelector('ul')

console.log(ul.firstChild)

Devuelve

<li>Soy el primer hijo<li>

lastChild

Devuelve el último nodo hijo. Ejemplo:

En HTML:

<ul>

<li>Soy el primer hijo<li>

<li>Soy el último hijo<li>

</ul>

En javascript:

const ul = document.querySelector('ul')

console.log(ul.firstChild)

Devuelve

<li>Soy el último hijo<li>

lastChild

Devuelve el nodo previo en la lista de nodos hermanos.

lastChild

Devuelve el nodo siguiente en la lista de nodos hermanos.
